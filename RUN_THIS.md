# ✅ RUN THESE 2 COMMANDS

## Command 1 - Backend (Terminal 1)
```bash
pnpm dev:server
```

Wait for: `[api] server listening on port 3000`

---

## Command 2 - Frontend (Terminal 2)
```bash
pnpm dev:metro
```

Wait for: Metro bundler to start, then open http://localhost:11000

---

## Done! ✅

- Backend: http://localhost:3000
- Frontend: http://localhost:11000

---

**If backend errors:** Share the error message  
**If frontend shows "Failed to fetch":** Make sure backend is running first
